## 公司简介

拉玛羊驼是一家围绕生成式AI的应用公司，在2023年底获得奇绩创坛陆奇博士，酷我音乐创始人雷鸣等人的种子轮投资。拉玛羊驼主要关注泛娱乐方向。

## 项目交付

平均每个网站一天之内完成研究、设计、编码与 demo，交付形式为 代码（基于 github） + API 文档（基于 apifox）。
